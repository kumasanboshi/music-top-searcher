## Issue #3: ランキングデータの収集・作成
**ラベル**: data

### 概要
1975〜2025年の邦楽・洋楽 年間TOP100ランキングデータを収集し、JSONファイルとして作成する。

### タスク
- [ ] データソースの選定（RIAJ、Wikipedia等の公開情報）
- [ ] データ収集の方針決定
  - ~2004年: CD売上ベース
  - 2005〜2015年: 過渡期（CD売上中心）
  - 2016年〜: 複合指標（ストリーミング + DL + CD）
- [ ] 邦楽ランキングデータの収集・入力（50年分）
- [ ] 洋楽ランキングデータの収集・入力（50年分）
- [ ] 収録CD情報（シングル・アルバム）の付与
- [ ] Amazon / Apple Music リンクの付与
- [ ] データのバリデーション（重複チェック、欠損チェック等）

### 法的留意事項
- 複数の公開情報源を参照し、独自基準で編集する
- 特定ランキングサービスのデータをそのまま転載しない
- 「当サイト独自の集計によるランキングです」と明記する

### 備考
最もボリュームの大きい作業。段階的に進める（例: まず数年分のサンプルデータを作成し、アプリ開発と並行）。
