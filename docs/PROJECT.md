# Music Top Searcher - プロジェクト仕様書

## 概要

過去50年分（1975〜2025年）の日本国内における邦楽・洋楽の年間売上TOP100を検索・閲覧できるWebアプリケーション。

## プラットフォーム

- **種別**: Webアプリ（静的サイト）
- **ホスティング**: Cloudflare Pages または Vercel（無料枠）
- **PWA対応**: 初回アクセス後はオフラインでも閲覧・検索可能

## データ仕様

### ランキングデータ

| 項目 | 内容 |
|------|------|
| 対象期間 | 1975年〜2025年（50年分） |
| ジャンル | 邦楽 / 洋楽 |
| ランキング数 | 各年・各ジャンル TOP100 |
| 総曲数 | 最大 10,000曲（50年 × 2ジャンル × 100曲） |
| データソース | 複数の公開情報源から独自に集計・編集したランキング |
| 保持形式 | JSONファイルとしてアプリ内にバンドル（ローカルデータ） |

### CD売上 / 配信の扱い

- 年ごとに1つのランキングとする（CD・配信で分けない）
- ラベル表示も不要
- 目安として以下の区分でデータを収集する:
  - ~2004年頃: CD売上ベース
  - 2005〜2015年頃: 過渡期（CD売上中心）
  - 2016年頃〜: 複合指標（ストリーミング + DL + CD）

### データ量見積もり

- JSON: 約3MB
- gzip圧縮後: 約500KB〜1MB
- 初回アクセス時に全データ取得（画像1枚程度の軽さ）

## 画面構成・UI

### 画面遷移

```
トップページ
├── 検索（アーティスト名・曲名）
├── 邦楽を選択
│   ├── 年別選択（1975, 1976, ... 2025）
│   │   └── ランキング一覧（TOP100）
│   │       └── 曲の詳細ページ
│   └── 年代別選択（70年代後半, 80年代, 90年代, 00年代, 10年代, 20年代）
│       └── ランキング一覧（TOP100）
│           └── 曲の詳細ページ
└── 洋楽を選択
    └── （同上）
```

### トップページ

- 邦楽 / 洋楽の選択
- アーティスト名・曲名の検索バー（TOP100に入った曲のみが対象）

### ランキング一覧ページ

- 年別または年代別のTOP100を表示
- 年代別の場合は複数年をまとめて表示

### 曲の詳細ページ

- 曲名、アーティスト名、ランキング順位、年
- 収録CD情報（シングル、アルバム）
- 外部リンク:
  - Amazon（CD / デジタル購入）
  - Apple Music
- 同アーティストの他のランクイン曲へのリンク（TOP100に入ったもののみ）

## 検索機能

- アーティスト名・曲名でのテキスト検索
- 検索対象はTOP100にランクインした曲のみ
- オフラインで動作（ローカルデータを検索）

## 収益化

| 手段 | サービス |
|------|---------|
| 広告 | Google AdSense |
| アフィリエイト | Amazonアソシエイト（CD / デジタル購入リンク） |
| アフィリエイト | Apple Performance Partners（Apple Musicリンク） |

※ Spotifyはアフィリエイトプログラムがないため対象外

## 非機能要件

| 項目 | 内容 |
|------|------|
| 言語 | 日本語のみ |
| オフライン | PWAとして対応（広告・アフィリエイトリンクはオンライン必要） |
| サーバーサイド処理 | なし（静的サイト） |
| 維持費 | 無料（ホスティング無料枠利用） |
| ドメイン | カスタムドメイン利用時のみ年間1,000〜2,000円程度 |

## 法的留意事項

- ランキングデータは複数の公開情報源を参照し、独自の基準で編集・作成する
- 特定のランキングサービス（オリコン等）のデータをそのまま転載しない
- サイト上に「当サイト独自の集計によるランキングです」と明記する
