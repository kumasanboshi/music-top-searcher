# PROJECT.md - プロジェクト詳細・要件

## 概要

**Music Top Searcher**（音楽トップサーチャー）は、1975年〜2025年の邦楽・洋楽ヒット曲TOP100ランキングを閲覧・検索できるWebアプリケーション。

## 機能一覧

### 1. トップページ
- アプリタイトル「Music Top Searcher」表示
- ジャンル選択: 邦楽（jpop）、洋楽（western）
- 検索ページへの導線「曲を検索」

### 2. 年・年代選択ページ
- 選択したジャンルのタイトル表示（例:「邦楽 - 年を選択」）
- **年代別選択**: 70年代後半、80年代、90年代、00年代、10年代、20年代（6つ）
- **年別選択**: 1975〜2025年（51個）

### 3. ランキング一覧ページ
- 年別ランキング: 指定年のTOP100を表示
- 年代別ランキング: 複数年のランキングを統合リストで表示（年情報付き）
- 各エントリに順位・曲名・アーティスト名を表示
- 曲名から詳細ページへのリンク

### 4. 曲詳細ページ
- 曲名、アーティスト名、ランキング順位・年
- CD情報（タイトル、種別、発売日）※データがある場合のみ
- 外部リンク（Amazon、Apple Music）※オンライン時のみ表示
- 同アーティストの他のランクイン曲

### 5. 検索ページ
- インクリメンタルサーチ（300msデバウンス）
- 検索対象: 曲名、アーティスト名、アーティスト英名
- 部分一致・大文字小文字無視
- 結果から曲詳細ページへのリンク

### 6. オフライン対応（PWA）
- Service Workerによるデータキャッシュ（CacheFirst、30日・最大1000件）
- オフライン時の警告バナー表示
- オフライン時は外部リンクを非表示

## 対象データ

| 項目 | 内容 |
|------|------|
| ジャンル | 邦楽（jpop）、洋楽（western） |
| 対象年 | 1975年〜2025年（51年分） |
| ランキング件数 | 各年・各ジャンルTOP100 |
| 年代区分 | late1970s, 1980s, 1990s, 2000s, 2010s, 2020s |
| ランキングファイル数 | 102件 |
| 楽曲詳細ファイル数 | 約10,000件 |

## デプロイ

- ホスティング: Cloudflare Pages
- CI/CD: GitHub Actions（push/PR to main）
- ビルド: `npx vite build`（tscを経由しない）
